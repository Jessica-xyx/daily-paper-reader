## 摘要
基于记忆的离散视觉-语言导航（VLN）智能体必须在部分可观测条件下行动，然而即使是强大的冻结骨干网络在测试时仍存在脆弱性。两种常见故障模式是记忆读取时历史证据的陈旧性以及动作选择时低效的局部回溯。我们提出DART-VLN，一种针对离散VLN的无训练测试时控制框架。DART-VLN结合了测试时记忆衰减（一种通过不重写存储内容来抑制陈旧和冗余证据的读取侧记忆重加权规则）与反循环正则化（一种在动作选择期间阻止立即反转的轻量级下一跳惩罚）。该框架不引入新的可学习参数，并保持学习到的骨干网络不变。在R2R和REVERIE上的实验展现出一致模式：仅使用衰减可在读取侧获得稳定增益，而衰减+反循环实现了最佳的整体质量-效率权衡，在关键设置下产生更短的轨迹、更低的运行时间和改进的导航性能。行为分析进一步证实，反循环正则化减少了局部回溯，并在冻结骨干网络下提高了路径效率。总体而言，结果表明适度的测试时控制可以在无需重新训练的情况下使基于记忆的离散VLN更加可靠和高效。

## Abstract
Memory-based discrete vision-language navigation (VLN) agents must act under partial observability, yet even strong frozen backbones remain vulnerable at test time. Two common failure modes are stale historical evidence at memory readout and inefficient local backtracking during action selection. We present DART-VLN, a training-free test-time control framework for discrete VLN. DART-VLN combines Test-Time Memory Decay, a read-side memory reweighting rule that suppresses stale and redundant evidence without rewriting stored content, with Anti-Loop Regularization, a lightweight next-hop penalty that discourages immediate reversals during action selection. The framework introduces no new learnable parameters and leaves the learned backbone unchanged. Experiments on R2R and REVERIE show a consistent pattern: decay-only provides stable read-side gains, while decay+anti-loop achieves the best overall quality-efficiency trade-off, yielding shorter trajectories, lower runtime, and improved navigation performance in key settings. Behavioral analysis further confirms that anti-loop regularization reduces local backtracking and improves path efficiency under frozen backbones. Overall, the results show that modest test-time control can make memory-based discrete VLN more reliable and efficient without retraining.

- **核心问题**：基于记忆的离散视觉-语言导航（VLN）智能体在测试时面临两大失效模式：
  1. **记忆读取时历史证据的陈旧性与冗余性**：随着导航进行，较早或重复观测到的记忆内容不再有用，却仍以相同强度参与决策，导致聚合噪声增加。
  2. **动作选择时的低效局部回溯**：智能体容易立即返回上一步位置或形成短循环，导致路径长度膨胀、步数浪费、运行时增加。

## 论文提出的方法论

- **核心思想**：在现有冻结骨干网络的推理循环内引入两种互补的轻量级控制机制，不引入新可学习参数，不修改模型权重。
- **关键技术细节**：

### 1. 测试时记忆衰减（Test-Time Memory Decay）
- **操作位置**：记忆读取侧（read-side），不重写存储内容。
- **轻量元数据**：为每个记忆槽维护三项元数据：
  - 槽年龄 $a_i = t - t_{\text{last}_i}$（最新刷新步距）
  - 访问次数 $c_i$
  - 新颖性 $n_i$：基于瞬时新颖度 $\nu_i = \text{clip}(1 - \cos(f^{\text{old}}_i, f^{\text{new}}_i), 0, 1)$ 的指数移动平均（$\rho=0.5$）
- **重加权公式**：
  \[
  w_i = \text{clip}\left( e^{-\lambda a_i} \cdot \left(1 - \alpha \frac{c_i}{c_i+1}\right) \cdot (0.5 + 0.5 n_i), \ w_{\min}, w_{\max} \right)
  \]
  - 第一项：**新鲜度项**，衰减陈旧槽
  - 第二项：**重复惩罚项**，抑制频繁访问的槽
  - 第三项：**新颖性偏好项**，鼓励近期仍有特征变化的槽
- **作用效果**：仅调整读取时各槽的贡献权重，使聚合记忆更干净。

### 2. 反循环正则化（Anti-Loop Regularization）
- **操作位置**：动作选择侧（action side），在argmax前对候选动作得分施加惩罚。
- **惩罚项**：
  \[
  p_t(v) = \beta_{\text{back}} \cdot \mathbb{I}[h(v) = v_{t-1}] + \beta_{\text{rev}} \cdot \mathbb{I}[\text{visit}(h(v)) \ge k]
  \]
  - 第一项：惩罚**立即回溯**（下一跳返回上一观测点）
  - 第二项（弱化）：惩罚**频繁重访**（下一跳访问次数 ≥ k = 2）
- **调整后得分**：$s'_t(v) = s_t(v) - p_t(v)$，然后argmax选动作。
- **范围**：仅用于测试时确定性推理，不修改学习到的骨干网络。

### 算法流程（Algorithm 1）
1. 初始化轨迹与访问计数。
2. 每步编码指令与当前观测，计算记忆权重（式3），聚合得到候选得分。
3. 对每个候选，计算下一跳并施加惩罚（式4），更新得分。
4. argmax选动作，执行，更新元数据，直到STOP或达到最大步数。

## 论文的主要结论与发现

- **保守的读侧记忆衰减（decay-only）** 单独即可稳定提升任务指标（如NE从2.83→2.59，R2R val unseen），同时降低运行时。
- **添加反循环正则化（decay+anti-loop）** 进一步实现了最佳整体质量-效率权衡：轨迹更短、运行时更低、SR/SPL/RGSPL在关键设置中提升。
- **写侧干预（update-only / full-mode）** 不够稳定，可能因扰动冻结骨干已学表示而效果变差。
- **行为分析**：反循环正则化的最直接效果是降低立即回溯率（如R2R从3.51%→2.01%，REVERIE从8.45%→5.99%），使路径更干净。
- **整体结论**：在被冻结的强记忆型导航器上，轻量的测试时控制可提升可靠性、效率，无需重训练。
