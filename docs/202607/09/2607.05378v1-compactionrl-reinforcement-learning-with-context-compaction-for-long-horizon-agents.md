## 摘要
长程智能体大语言模型日益受到有限上下文窗口的限制，因为扩展的交互轨迹可能在任务完成前超过最大上下文长度。上下文压缩通过总结之前的交互状态并在压缩上下文中继续生成轨迹提供了一种自然解决方案，但将压缩融入强化学习尚未得到充分探索。我们提出CompactionRL，一种利用上下文压缩训练长程智能体大语言模型的强化学习策略。该方法通过令牌级损失归一化和跨轨迹广义优势估计，联合优化任务执行与摘要生成。该设计使LLM智能体能够从压缩后的长程轨迹中学习。我们在开源模型基础上训练CompactionRL，并在智能体编码任务上观察到一致的性能提升。CompactionRL使开源GLM-4.5-Air模型（106B-A30B）在SWE-bench Verified上达到66.8%的Pass@1分数，在Terminal-Bench 2.0上达到24.5%，绝对提升分别为7.0和3.1个百分点。基于GLM-4.7-Flash（30B-A3B）构建的CompactionRL将Pass@1分别提升5.5和6.8个百分点，在SWE-bench Verified和Terminal-Bench 2.0上分别达到56.0%和20.2%。因此，CompactionRL已被部署在用于训练开源GLM-5.2模型（750B-A40B）的RL流程中。

## Abstract
Long-horizon agentic LLMs are increasingly limited by finite context windows, as extended interaction trajectories can exceed the maximum context length before a task is completed. Context compaction offers a natural solution by summarizing previous interaction states and continuing the rollout under a compressed context, but incorporating compaction into reinforcement learning remains underexplored. We propose CompactionRL, a reinforcement learning strategy to train long-horizon agentic LLMs with context compaction. Our approach jointly optimizes task execution and summary generation with token-level loss normalization and cross-trajectory generalized advantage estimation. This design enables the LLM agents to learn from compacted long-horizon trajectories. We train CompactionRL on top of open models and observe consistent performance gains on agentic coding tasks. CompactionRL enables the open GLM-4.5-Air model (106B-A30B) to achieve Pass@1 scores of 66.8% on SWE-bench Verified and 24.5% on Terminal-Bench 2.0, with absolute gains of 7.0 and 3.1 points, respectively. Built upon GLM-4.7-Flash (30B-A3B), CompactionRL improves Pass@1 by 5.5 and 6.8 points, reaching 56.0% on SWE-bench Verified and 20.2% on Terminal-Bench 2.0, respectively. CompactionRL is thus deployed in the RL pipeline for training the open GLM-5.2 model (750B-A40B).

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：长时域交互式LLM智能体在复杂任务（如软件工程、终端操作）中，随着交互步数增加，累积历史（工具输出、中间推理、错误信息等）容易超过模型的有限上下文窗口。虽然长上下文模型可以缓解，但扩展上下文长度成本高且存在“迷失在中间”问题。
- **现有方法局限**：已有的上下文压缩（如记忆、反思、提示压缩）多作为推理时启发式或外部记忆操作，未与强化学习训练结合。在RL训练中，压缩会改变轨迹结构（一个轨迹被分为多个段），导致组归一化不适用、损失加权偏倚和时间信用分配问题。
- **本文贡献**：提出CompactionRL，首次将可训练的上下文压缩融入PPO强化学习框架，联合优化任务执行和摘要生成，使智能体在固定上下文预算下有效学习长程轨迹。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：将上下文压缩作为训练 rollout 收集的一部分。当剩余上下文预算低于阈值时，策略生成一个紧凑摘要，然后从“摘要+最近k步”重构上下文继续交互。摘要和任务执行动作由同一可训练策略生成，共享最终任务奖励。
- **关键技术细节**：
  - **可训练的上下文压缩**：在 rollout 中，当上下文接近限制时，触发压缩（剩余预算 < T_comp），策略采样生成摘要。重构上下文保留摘要和最近k步观察（k=2）。摘要不是外部模块，是策略的一部分。
  - **训练段划分**：完整 rollout 被划分为多个段（执行段和摘要段），每个段是生成 token 的序列。所有段共享最终任务奖励，无需额外摘要质量奖励。
  - **令牌级损失归一化**：使用 token 级损失（每 token 计算 PPO 损失）而非序列级损失，避免因段数量和长度异质导致的偏差。每个 token 权重相等。
  - **跨轨迹广义优势估计（GAE）**：解决段间信用分配问题。对段 σ_s 内 token i，本地 GAE 计算 A_loc_s,i，然后乘以折扣因子 (γλ)^{N_>s}，其中 N_>s 是同一 rollout 中后续段的总 token 数，使早期段获得与真实时间距离匹配的折扣。
  - **优化框架**：基于 PPO，使用价值函数（critic）估计优势，避免组归一化对段数变化的敏感。采用长度自适应 GAE（λ=1-1/(αl), α=1.5），每批做2次价值更新+1次策略更新。

## 6. 主要结论与发现
- CompactionRL 在固定峰值上下文预算下，显著超越基座模型和无压缩RL模型。在 SWE-bench Verified 上：GLM-4.5-Air 达到66.8%（+7.0），GLM-4.7-Flash 达到56.0%（+5.5）；Terminal-Bench 2.0 分别提升3.1和6.8个百分点。
- 标准PPO（无压缩）提升单窗口性能，但无法一致迁移到压缩推理；CompactionRL 专门适应压缩环境。
- 优化摘要生成（将摘要包含在RL损失中）是关键：产生更长更详细的摘要，保留更多上下文，减少冗余探索，提升压缩任务上的准确率。
- 令牌级损失归一化和跨轨迹 GAE 均不可或缺，移除任一组件均导致性能下降，其中令牌级损失影响更大。

## 7. 优点
- **方法创新**：首次将可训练的上下文压缩集成到LLM智能体RL训练中，而非仅作为推理时启发式。
- **处理段异质性**：通过 token 级损失消除段数/长度偏差，通过跨轨迹 GAE 解决跨段信用分配，方案设计精巧。
- **一致增益**：在两个模型规模、两个基准上均观察到显著提升，且训练成本增加有限（固定上下文预算）。
- **实际部署**：已用于训练更大的 GLM-5.2 模型，说明方法可扩展和实用。
- **分析充分**：提供压缩次数、工具调用、摘要长度、推理 token 等行为分析，解释性能来源。
