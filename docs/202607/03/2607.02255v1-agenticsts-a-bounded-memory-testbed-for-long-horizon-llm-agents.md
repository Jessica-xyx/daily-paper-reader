## 摘要
记忆对于长程LLM智能体而言，是一个关于每个未来决策允许看到什么内容的契约。最简单的契约将过去的观察、工具调用和反思追加到每个提示中，这使得先前的上下文易于访问，但也将其变成一种混杂的混合物，其中任何单个记忆成分的效果都难以隔离。我们引入并实现了一种替代的有界契约：每个决策都由通过类型化检索组装的全新用户消息来完成，不附加原始的跨决策转录。因此，提示在任何长度的运行中保持有界，并且任何单层都可以被独立消融。我们在《杀戮尖塔2》中实例化该契约，这是一个规则封闭的随机卡组构建游戏，其运行需要数百个战术和战略决策。同一游戏的前沿LLM的公开在线基准测试报告称，在最低难度下，五种配置的胜率为零，而开发者报告的人类在同一难度下的胜率为16%；任务困难但尚未饱和。在我们的框架内，当触发战略技能时，固定A0消融显示出最大的观察差异：无存储基线在10场游戏中获胜3场，添加技能层后获胜6场。在此样本量下，比较是指向性的而非统计上决定性的（Fisher精确检验p≈0.37）；跨骨干探测和公共累积上下文基线作为操作性比较报告，而非对契约变量本身的受控测试。我们发布了一个可复现的测试平台：298条带有条件标签的完整轨迹、冻结的记忆/技能快照、提示记录和分析脚本——这是一种智能体设计和一种经过验证的、可重复使用的方法，用于研究显式记忆层如何塑造长程LLM智能体的决策。

## Abstract
Memory for a long-horizon LLM agent is a contract about what each future decision is allowed to see. The simplest contract appends past observations, tool calls, and reflections to every prompt, which makes prior context easy to access but also turns it into a jumbled mixture in which the effect of any single memory component is hard to isolate. We introduce and instrument an alternative bounded contract: every decision is made from a fresh user message assembled by typed retrieval, with no raw cross-decision transcript appended. The prompt thus stays bounded across runs of any length, and any single layer can be ablated in isolation. We instantiate the contract in Slay the Spire 2, a closed-rule stochastic deck-building game whose runs require hundreds of tactical and strategic decisions. A public online benchmark of frontier LLMs on the same game reports zero wins at the lowest difficulty across five configurations, and the developer-reported human win rate at the same difficulty is 16%; the task is hard but not saturated. Within our harness, a fixed-A0 ablation shows the largest observed difference when triggered strategic skills are enabled: the no-store baseline wins 3/10 games and adding the skill layer 6/10. At this sample size the comparison is directional rather than statistically decisive (Fisher exact p\approx0.37); a cross-backbone probe and public accumulating-context baselines are reported as operational comparisons rather than controlled tests of the contract variable itself. We release a reproducible testbed: 298 completed trajectories with condition tags, frozen memory/skill snapshots, prompt records, and analysis scripts -- an agent design and a validated, reusable methodology for studying how explicit memory layers shape long-horizon LLM-agent decisions.

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想：有界记忆契约（Bounded-Memory Contract）**
    - 该方法摒弃了“拼接式”的做法，不将历史对话记录附加到每个决策的提示中。相反，**每个决策都由一个全新合成的用户消息（user message）驱动**，该消息是从五个**类型化（typed）**的知识层中通过检索组装而成。
    - 这个设计的核心优势在于：① **有界性（Bounded）**：提示大小在任意长度的运行中都保持近似恒定，不随决策数量增长。② **可消融性（Ablatable）**：每个单一的记忆层（如规则、经验、策略）都可以被独立地开启或关闭，从而精确评估其贡献。

- **关键技术细节：五层记忆架构**
    - 该方法构建了一个名为 **AgenticSTS** 的智能体，其决策提示由以下五个分层构成：
        1.  **L1 - 协议指令（Protocol Instructions）**：固定不变的角色和协议模板。
        2.  **L2 - 状态模式（State Schemas）**：固定不变的用于描述当前状态和合法动作格式的模式。
        3.  **L3 - 游戏知识（Game Knowledge）**：从数据库检索的、可枚举的游戏规则库（卡牌、遗物、敌人等）。该层是可过滤的。
        4.  **L4 - 片段记忆（Episodic Memory）**：存储过往回合的总结（如“对某Boss的战斗策略”），供后续检索。该层可开关或冻结。
        5.  **L5 - 技能库（Skill Library）**：存储由环境触发的战略性指导（如“在何种情况下优先防守”）。这些技能可以是人类专家编写（Mode A），也可以由模板自动填充（Mode B）或通过事后分析（postrun analysis）从日志中蒸馏而来。该层可开关或冻结。

- **关键流程（文字描述）**：
    1.  **输入**：当前游戏状态 `sd`。
    2.  **检索**：根据当前状态，从L1到L5五个层中检索相关信息（L1和L2是固定的，L3-L5需要检索）。
    3.  **组装**：将检索到的五个切片按照固定顺序组合成一个全新的用户消息 `ud`。
    4.  **决策**：将 `ud` 和系统提示（`sys`）一起发送给LLM，LLM据此做出决策。
    5.  **更新**：对于L4和L5层，可以在单次回合（run）结束后，通过分析本轮日志（postrun extraction and skill discovery）来更新其内容。**每次决策的原始对话记录不会被附加到下一次决策中**。

### 6. 论文的主要结论与发现

1.  **有界记忆契约使记忆实验成为可能**：最主要的结论是，作者设计的类型化、有界的记忆接口确实可以被当作一个**可实验、可消融的变量**来评估，这超越了将记忆简单地视为“文本存储”的常规做法。
2.  **技能层（L5）是主要贡献者**：在固定A0的消融实验中，最大的性能提升（胜率从3/10提升至6/10）源于添加了L5战略性技能层。虽然统计上还不显著，但这指明了未来研究的重点方向：是**触发式、高层次的技能指导**比“多看几轮”的历史更重要。
3.  **片段记忆层（L4）在A0难度下饱和**：L4的加入并没有在A0难度下带来额外提升，但其在应对更高难度（A6-A8）的阶梯中有潜在价值。
4.  **记忆堆栈对底层模型敏感**：一个在Gemini上训练好的L4+L5记忆堆栈，在迁移到Qwen时能提升分数，但对DeepSeek却是负作用。这表明**记忆需要与底层模型能力对齐**，不能“一刀切”。
5.  **“有界”比“拼接”更高效**：在与开源智能体的直接对比中，作者的有界记忆智能体（甚至是最基础的baseline）不仅在游戏得分上远超使用累积上下文的智能体，而且速度和token成本也低一至两个数量级。这证明了该架构在面对长程决策时具有显著的工程效率和效能优势。
6.  **任务未饱和**：当前任务对LLM智能体而言仍然极具挑战性（外部模型胜率为0%，人类为16%），这为未来研究提供了空间。

- **核心方法论创新**：将记忆从一个被动、混杂的“文本箱”转化为一个**主动、可分离、可定制的契约（L1-L5层结构）**。这是该工作的最大亮点，提供了清晰的因果归因路径。

- **样本量和统计推断能力**：这是最显著的局限性。小样本量（N=10）导致结果不显著，无法对“哪种记忆层更重要”给出强有力的确认。
- **单一游戏和角色**：所有实验均基于《杀戮尖塔2》和“寂静（Silent）”角色（Silent是修改过的英文，应为 The Silent）。其结论的泛化性到其他游戏或不同类型的AI任务（如网页导航、编程）是需要验证的。
- **外部比较的公平性**：虽然尽力公平，但与开源智能体的对比本质上仍是“工程系统”间的比较，而非严格隔离了“记忆契约”变量的A/B测试。
- **训练无关性**：本文中心是免训练的推理架构，未涉及对LLM进行微调。训练好的记忆是否能通过微调进一步改善是一个开放问题。
- **技能的来源与质量**：L5层技能的来源（专家编写和模板填充）限制了其自主性和创造性。“Mode B”自动模板填充只测量了其与“Mode A”竞争的初步结果，大部分技能仍被拒绝或合并，表明真正的自动发明技能仍需探索。
- **计算开销**：虽然“有界”契约在推理时节省Token，但每个决策仍需进行多次检索，这种检索本身是有计算成本的（尽管比拼接整个历史小得多）。
