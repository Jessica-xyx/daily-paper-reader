## 摘要
LLM智能体能否在没有梯度更新的情况下，通过自我生成的记忆来改进决策？我们提出了FORGE（失败优化反思性毕业与进化），一种分阶段的、基于群体的协议，用于演化注入提示的自然语言记忆，以支持层次化ReAct智能体。FORGE包含一个反思式内循环，其中专门的反思智能体（使用相同的底层LLM，无需从更强模型进行蒸馏）将失败轨迹转化为可重用的知识工件：文本启发式（规则）、少样本演示（示例）或两者兼有（混合）。外循环在阶段之间将表现最佳的实例记忆传播到群体中，并通过毕业标准冻结收敛的实例。我们在CybORG CAGE-2上进行了评估，这是一个随机网络防御POMDP，在30步时间跨度内对抗B-line攻击者。所有测试的四个LLM家族（Gemini-2.5-Flash-Lite、Grok-4-Fast、Llama-4-Maverick、Qwen3-235B）都表现出强烈的负值、重尾零样本奖励。与零样本基线和反思基线（孤立单流学习）相比，FORGE在所有12个模型-表示条件下，平均评估回报提升了零样本的1.7-7.7倍，比反思提升了29-72%，将重大失败率（低于-100）降低至约1%。我们发现：（1）群体广播是关键机制，无毕业消融实验证实广播带来了性能提升，而毕业主要节省计算；（2）对于四个模型中的三个，示例取得了最强的回报，规则提供了最佳的成本-可靠性曲线，令牌数减少约40%；（3）较弱的基线模型受益不成比例，表明FORGE可能缩小能力差距而非放大强模型。所有证据限于CAGE-2 B-line；跨家族发现是指向性的证据。

## Abstract
Can LLM agents improve decision-making through self-generated memory without gradient updates? We propose FORGE (Failure-Optimized Reflective Graduation and Evolution), a staged, population-based protocol that evolves prompt-injected natural-language memory for hierarchical ReAct agents. FORGE wraps a Reflexion-style inner loop, where a dedicated reflection agent (using the same underlying LLM, no distillation from a stronger model) converts failed trajectories into reusable knowledge artifacts: textual heuristics (Rules), few-shot demonstrations (Examples), or both (Mixed), with an outer loop that propagates the best-performing instance's memory to the population between stages and freezes converged instances via a graduation criterion. We evaluate on CybORG CAGE-2, a stochastic network-defense POMDP at a 30-step horizon against the B-line attacker, where all four tested LLM families (Gemini-2.5-Flash-Lite, Grok-4-Fast, Llama-4-Maverick, Qwen3-235B) exhibit strongly negative, heavy-tailed zero-shot rewards. Compared against both a zero-shot baseline and a Reflexion baseline (isolated single-stream learning), FORGE improves average evaluation return by 1.7-7.7$\times$ over zero-shot and by 29-72% over Reflexion in all 12 model-representation conditions, reducing major-failure rates (below $-100$) to as low as $\sim$1%. We find that (1) population broadcast is critical mechanism, with a no-graduation ablation confirming that broadcast carries the performance gains while graduation primarily saves compute; (2) Examples achieves the strongest returns for three of four models, Rules offers the best cost-reliability profile with $\sim$40% fewer tokens; and (3) weaker baseline models benefit disproportionately, suggesting FORGE may mitigate capability gaps rather than amplify strong models. All evidence is confined to CAGE-2 B-line; cross-family findings are directional evidence.

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想：FORGE协议

FORGE（Failure-Optimized Reflective Graduation and Evolution）是一个**分阶段、基于种群**的协议，它通过**内循环-外循环**结构演化注入到提示中的自然语言记忆：
- **内循环**（Reflexion风格）：智能体执行回合，若单步奖励低于阈值则触发反射，专用学习器（与行动LLM相同，无更强模型蒸馏）分析失败轨迹并生成知识工件（规则或示例），追加到记忆后重启回合。
- **外循环**（种群广播与毕业）：多个智能体实例并行运行；每阶段后，选取表现最好的**冠军**实例，将其完整记忆广播给所有活跃实例；满足毕业阈值的实例被冻结（毕业），退出训练以节省算力。

### 2.2 关键技术细节

#### 2.2.1 分层ReAct智能体架构（图1左）
- **顶层规划器（Planner）**：选择最终防御动作，可委托两个子智能体（作为工具实时调用）：
  - **分析师（Analyst）**：解释主机级观测。
  - **动作选择器（ActionChooser）**：对有效动作进行排序并给出理由。
- 所有子智能体使用相同底层LLM，仅系统提示和注入的记忆不同。
- 记忆分为**动态记忆**（训练中由学习器生成）和**持久记忆**（用户设定，如动作参考表）。

#### 2.2.2 内循环：失败触发反射（Algorithm 1）
- 每步奖励 $r_{step}$ 若低于故障阈值 $\tau = -1.1$，则立即中止回合，捕获完整轨迹和当前记忆。
- 专用学习器（Reflector生成规则，Exemplifier生成示例）基于轨迹合成新知识工件。
- 工件追加到记忆，回合从第0步重新开始。每阶段最多重复 $k_A = 3$ 次。

#### 2.2.3 外循环：FORGE协议（Algorithm 2）
- 参数：$N=10$ 个并行实例，$S=6$ 个阶段，$k_A=3$ 尝试/阶段，毕业阈值 $\theta=-15$。
- **阶段同步**：所有实例完成内循环后，进行检查点评估（冻结，无学习）。
- **冠军选择**：从活跃（未毕业）实例中选择检查点回报最高的实例。
- **广播**：冠军的完整内存替换所有其他活跃实例的内存（全替换，不可插值）。
- **毕业**：检查点回报 $> -15$ 的实例被冻结，排除出后续阶段。若禁用广播，协议退化为并行独立Reflexion（即基线）。

#### 2.2.4 记忆表示
三种表示作为实验变量：
- **规则（Rules）**：条件式启发列表（如“当横向移动确认时，计划：隔离路径并分析相邻主机”）。
- **示例（Examples）**：完整的ReAct交互演示（Thought-Tool-Observation-Answer序列）。
- **混合（Mixed）**：同时对同一上下文生成规则和示例。

## 6. 主要结论与发现

1. **巨大性能提升**：FORGE在所有12种模型-表示组合上均优于零样本和Reflexion：
   - 相比零样本：提升1.7–7.7倍（Gemini从-189.6到-24.5）
   - 相比Reflexion：提升29–72%
2. **种群广播是关键机制**：无毕业消融同样优于Reflexion，确认广播带来性能提升；毕业主要贡献计算节省（减少活跃实例数及令牌消耗）。
3. **记忆表示对比**：
   - **示例（Examples）**：在4个模型中的3个取得最佳平均回报（如Gemini -24.5）
   - **规则（Rules）**：成本-效益比最佳，令牌消耗比示例少约40%，且毕业率更高
   - **混合（Mixed）**：居中，无显著优势
4. **弱模型受益更大**：基线越差的模型提升倍数越高（Gemini 7.7×，Qwen 4.3×，Llama 4.0×，Grok 1.7×），表明FORGE主要缩小能力差距而非放大强模型。
5. **尾风险大幅降低**：零样本约90%回合低于-100，最强FORGE配置降至约1%。
6. **峰值性能**：单个检查点回报达到-3.60，接近DRL最佳-3.47。
7. **阈值非最优**：灵敏度实验发现τ=-11.0（仅严重失败触发）优于默认τ=-1.1，说明设计空间可进一步探索。

## 7. 优点

- **完全无梯度更新**：无需微调或额外模型蒸馏，仅通过上下文中文本工件的演化即可实现策略提升，非常适合API受限场景。
- **内循环+外循环协同设计**：内循环利用失败信号生成有针对性的知识，外循环通过冠军广播施加选择压力，有效抑制了孤立Reflexion的高方差和退化问题。
- **结构化知识表示比较**：系统对比了规则、示例、混合三种表示在随机长时间跨POMDP中的效率，为后续研究提供了实证参考。
- **毕业机制聪明地节省算力**：冻结强解避免后期广播覆盖退化，同时减少活跃实例数，但消融表明其贡献主要在成本而非性能。
- **跨模型验证**：在4个不同家族、不同能力的模型上均取得一致正向效果，增强了方法的通用性。
- **失效阈值基于奖励分布分析**：通过540个零样本回合的奖励分布，选择τ=-1.1实现100%精确率（排除合法Restore操作的-1.0），避免误触发。

## 8. 不足与局限

1. **环境单一**：所有实验仅在CybORG CAGE-2的B_line攻击者、30步时间跨度下进行，作者明确承认未泛化到其他攻击者类型或POMDP环境。
2. **跨模型结论为方向性**：非Gemini模型仅3-4组会话，统计强度不足，作者称之为“directional evidence”。
3. **阈值选择非最优**：灵敏度实验发现τ=-11.0（仅最严重失败）效果更好，而论文主实验采用τ=-1.1，可能低估了FORGE的潜在上限。
4. **广播策略粗糙**：采用“单最佳替换”的硬方式，可能丢失种群多样性；元朗研究也提到冲突解析的不可靠性限制了插值等更精细的混合方式。
5. **与前沿方法对比不足**：虽以Reflexion为基线合理，但未与TextGrad、Dynamic Cheatsheet、ACE等更近期的提示优化方法对比（作者承认这是未来工作）。
6. **计算成本报告不完整**：未披露API调用总成本或实际运行时间，仅报告令牌数量；不同模型间的令牌价格差异未考虑。
7. **仅支持标量奖励**：内循环依赖单步奖励阈值触发，对于提供自然语言反馈的环境（如Voyager）可能无法直接迁移。
8. **结论局限性**：所有发现限于特定基准，向通用LLM自我改进方法的推广需谨慎。
