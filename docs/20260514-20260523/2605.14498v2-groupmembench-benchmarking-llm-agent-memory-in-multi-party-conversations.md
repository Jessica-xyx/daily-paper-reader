## 摘要
大型语言模型智能体越来越多地用作个人助手和工作场所协作工具，其效用依赖于能够在长时间对话中提取、检索和应用信息的记忆系统。然而，现有的记忆系统和基准测试都是围绕二元、单用户设置构建的，而实际部署通常涉及多个用户与智能体以及彼此之间相互交互的群组和频道。这种不匹配导致群体记忆的三个特性未被测量：（i）超越简单拼接一对一聊天的群体动态，（ii）基于说话者的事实追踪，需要为每个用户建立记忆模型，以及（iii）适应听众的语言，其中心理理论转移导致角色特定的词汇。我们提出了GroupMemBench，一个暴露所有这三个特性的基准测试。一个基于图结构的数据合成流程生成具有可控回复结构的多轮对话，并根据每个用户的人设和目标受众生成每条消息。一个对抗性查询流程将每个问题绑定到六类中的特定提问者，涵盖多跳推理、知识更新、术语歧义、用户隐含推理、时间推理和弃权，并迭代搜索反映综合记忆能力的具有挑战性的现实查询。对领先记忆系统的基准测试揭示了明显的崩溃：最强的系统仅达到46.0%的平均准确率，知识更新为27.1%，术语歧义为37.7%，而简单的BM25基线匹配或超过了大多数智能体记忆系统。这表明当前的记忆摄入过程抹去了群体记忆所依赖的结构和词汇特征，使得多用户记忆远未得到解决。

## Abstract
Large Language Model (LLM) agents increasingly serve as personal assistants and workplace collaborators, where their utility depends on memory systems that extract, retrieve, and apply information across long-running conversations. However, both existing memory systems and benchmarks are built around the dyadic, single-user setup, even though real deployments routinely span groups and channels with multiple users interacting with the agent and with each other. This mismatch leaves three properties of group memory unmeasured: (i) group dynamics that go beyond concatenated one-on-one chats, (ii) speaker-grounded belief tracking, where the per-user memory modeling is needed, and (iii) audience-adapted language, where Theory-of-Mind shifts produce role-specific vocabulary. We introduce GroupMemBench, a benchmark that exposes all three. A graph-grounded synthesis pipeline produces multi-party conversations with controllable reply structure and conditions each message on per-user personas and target audiences. An adversarial query pipeline then binds every question to a specific asker across six categories, spanning multi-hop reasoning, knowledge update, term ambiguity, user-implicit reasoning, temporal reasoning, and abstention, and iteratively searches challenging, realistic queries that reflect comprehensive memory capability. Benchmarking leading memory systems exposes a sharp collapse: the strongest one reaches only 46.0% average accuracy, with knowledge update at 27.1% and term ambiguity at 37.7%, while a simple BM25 baseline matches or exceeds most agent memory systems. This indicates current memory ingestion erases the structural and lexical features group memory depends on, leaving multi-user memory far from solved.

- **研究动机**：大型语言模型（LLM）智能体被广泛用作个人助手和协作工具，其效用依赖于从长时间对话中提取、检索和应用信息的记忆系统。然而，现有记忆系统和基准均针对二元（dyadic）、单用户对话设计，而实际部署中智能体常处于多用户群聊或频道环境。这种不匹配导致三个群体记忆关键特性未被评估：① **群体动态**（超越单对一对话的线程化讨论、跨主题切换等）；② **基于说话者的事实追踪**（不同用户说的话需要区分记忆，同一问题因提问者身份不同答案可能不同）；③ **受众适应语言**（说话者会基于心理理论调整词汇，例如工程师用“PPO”而管理者说“对齐”）。

## 6. 论文的主要结论与发现

- **核心发现**：当前所有记忆系统在GroupMemBench上表现不佳，最强方法Hindsight仅达46.0%平均准确率。知识更新（27.1%）和术语歧义（37.7%）尤其差。简单BM25基线准确率43.2%，超过大多数智能体记忆系统，表明现有人工记忆摄入过程损失了对群体记忆至关重要的结构和词汇特征。
- **具体观察**：
  - 群体结构在摄入时被压平：多跳推理准确率低于50%，不同方法差异小。
  - 记忆系统不区分说话者：用户隐含推理、知识更新、术语歧义均表现差，说明缺乏说话者条件建模。
  - 词汇歧义是检索成功后的主要障碍：即使找到支持消息，LLM也难以基于不同说话者的不同用语正确推理。
  - 从性能-效率角度看，大部分系统被HippoRAG帕累托支配，而BM25（零摄入成本）严格帕累托支配多数智能体系统。
  - 失败分析显示：检索失败是主要瓶颈（占错误的41-79%），推理失败仅占7-24%；但术语歧义类在检索成功后仍错误率高，说明记忆表征本身有问题。

## 7. 优点

- **创新性**：首次明确提出并定义了多用户群体记忆的三个关键维度（动态、说话者接地、受众适应），并设计了对应的评估方法。
- **严谨性**：采用图驱动合成对话，可控性好且贴近真实；使用对抗性查询管道确保问题有难度，避免简单匹配。
- **公平比较**：统一底层模型，区分摄入和查询阶段，区分检索和推理失败，提供成本/存储指标，使分析全面。
- **实践意义**：指出当前系统本质问题（摄入抹杀了结构/词汇特征），为未来多用户记忆系统设计提供了明确方向。

## 8. 不足与局限

- **数据局限性**：仅基于合成对话，可能无法完全代表真实世界多用户交互的多样性和非正式性；仅覆盖四个领域（科技、金融、医疗、制造），且为英文。
- **评估局限性**：仅采用准确率作为主要指标，忽略了部分回答正确但冗余或不够精确的情况；LLM judge虽经检验，但仍有近1%的偏差（略严格），可能低估实际性能。
- **方法覆盖不足**：未测试多模态（图片、附件）、多语言或具有更复杂推理（如计划、工具使用）的场景；仅测试了基于记忆的问答，未测试交互过程中的记忆更新。
- **系统测试范围**：只测试了公开可用的记忆系统，不包括商业化或最新未公开的系统；BM25之所以表现好可能是因为其简单的原始文本检索保留了所有结构，但实际应用可能受限于效率或其他因素。
- **未涉及可控实验**：没有对记忆系统进行修改以验证其缺陷可被修复；也没有探讨不同基线在多个随机种子下的稳定性（除质量评估外）。
