## 摘要
长期运行的语言智能体需要的不仅仅是记忆访问。检索系统可以在查询时获取过去的事实，但它们不会决定在工作上下文被卸载后哪些经验应继续塑造行为。我们将这个独立的问题研究为记忆深度：持久的目标导向倾向被写入一个小型参数化存储中。我们引入了循环漂移协议（loop-drift protocol），这是一个受控的压力测试，其中检索索引保持不变，同时工作上下文被卸载，目标导向行为必须在长循环干扰下持续存在。我们评估了EVAF，一种基于惊奇和效价门控的LoRA巩固机制。在GPT-2和TinyLlama上，检索在浅层事实回忆方面最强（短事实准确率0.956--0.973），而EVAF在目标持久性和卸载后的恢复方面最强（0.812--0.904），每200个事件仅进行2-3次参数化写入。机制控制表明，选择性巩固可以分解为两个可控维度：选择（selection）和驱动（actuation）。匹配的随机门控隔离了稀疏写入之外的选择；在GPT-2、TinyLlama和Mistral-7B上的固定内循环控制表明，内循环写入强度依赖于模型；而Mistral-7B的匹配门控反转揭示了在驱动校准不当的情况下不对称的选择-驱动耦合。公开的Memora事件流作为外部诊断，暴露了陈旧记忆失效（stale-memory invalidation）作为一个未解决的边界。在这个探测中，选择性参数巩固提供了与检索访问不同且互补的记忆深度。

## Abstract
Long-running language agents need more than memory access. Retrieval systems can fetch past facts at query time, but they do not decide which experiences should continue to shape behavior after the working context is unloaded. We study this separate problem as memory depth: durable goal-conditioned tendencies written into a small parametric store. We introduce the loop-drift protocol, a controlled stress test in which the retrieval index remains intact while working context is unloaded and goal-conditioned behavior must persist under long-loop interference. We evaluate EVAF, a surprise- and valence-gated LoRA consolidation mechanism. Across GPT-2 and TinyLlama, retrieval is strongest on shallow factual recall (short-fact accuracy 0.956--0.973), while EVAF is strongest on goal persistence and post-unload recovery (0.812--0.904) with only 2--3 parametric writes per 200 events. Mechanism controls show that selective consolidation factorizes into two controllable dimensions: selection and actuation. Matched random gates isolate selection beyond sparse writing; fixed-inner controls across GPT-2, TinyLlama, and Mistral-7B show that inner-loop write strength is model-dependent; and a Mistral-7B matched-gate inversion reveals asymmetric selection-actuation coupling under miscalibrated actuation. Public Memora event streams serve as an external diagnostic, exposing stale-memory invalidation as an unresolved boundary. Within this probe, selective parametric consolidation supplies memory depth distinct from and complementary to retrieval access.

- **整体含义**：提出选择性参数巩固（selective parametric consolidation）作为记忆深度的实现机制，并证明它能与检索互补，共同满足长期代理的记忆需求。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：设计一个名为 **EVAF**（Surprise- and Valence-gated LoRA consolidation）的机制，通过**惊奇门控**和**效价门控**，仅选择行为相关的事件写入一个小型可参数化存储（LoRA适配器），实现低开销、高持久性的目标行为巩固。
- **关键技术细节**：
  - **事件选择门**：对每个事件 $x_t$，计算：
    - 惊奇分数 $s_t$：基于 token 负对数似然（模型预测的不确定性）。
    - 效价分数 $v_t$：基于事件嵌入与用户持久目标/偏好的嵌入相似度。
    - 写出准入分数：$g_t = \sigma(k_s(s_t - \tau_s)) \cdot \sigma(k_v(v_t - \tau_v))$，其中 $\sigma$ 是 sigmoid 函数，$\tau_s, \tau_v$ 为阈值，$k_s, k_v$ 为斜率。
    - 若 $g_t > \tau_w$，则事件进入小缓冲区。
  - **适配器更新**：当缓冲区填满（固定大小），使用来自缓冲区及先前巩固的重放样本更新 LoRA 模块，并加入 L2 锚定（anchor）以防止漂移。
  - **驱动（Actuation）控制**：将选择与写入强度解耦。通过固定内循环步数（Fixed-1/2/3）或行为边际控制器（Margin）调节写力量级。
- **算法流程（文字描述）**：
  1. 初始化 LoRA 适配器（参数为零，锚定在初始值）。
  2. 遍历用户事件流，对每个事件计算 $g_t$。
  3. 若 $g_t > \tau_w$，将该事件加入缓冲区；否则跳过。
  4. 当缓冲区达到大小下限，执行 LoRA 更新：在缓冲区数据 + 重放样本上微调，同时最小化 $||\theta - \theta_0||^2$（L2 锚定）。
  5. 清空缓冲区，继续后续事件。
  6. 在评估时，使用更新后的模型计算目标行为得分（匹配续写评分，higher is better for goal/persist probes）。

### 6. 论文的主要结论与发现

1. **深度翻转（Depth Flip）**：检索（RAG）在浅层事实回忆上显著优于 EVAF（短事实准确率 0.956–0.973），而 EVAF 在目标持久性和卸载后恢复上大幅领先（0.812–0.904），每 200 事件仅需 2–3 次参数写入。
2. **选择性参数巩固不同于稀疏性**：匹配随机门控的实验证明，选择特定事件（而非仅仅少写）是关键。在 GPT-2 上 EVAF 明显优于随机匹配门控；但在 TinyLlama 上效果弱化，在 Mistral-7B 上甚至反转（因驱动过强），暴露了选择与驱动的耦合。
3. **选择与驱动是可分离的因子**：固定内循环步数（Fixed-1/2/3）和匹配门控实验表明，写入强度是独立的控制维度，且在不同模型上最优设置不同（GPT-2 以 Fixed-1 最佳，Mistral-7B 以 Fixed-2 可恢复稳定）。
4. **互补性验证**：路由 EVAF+RAG 在 GPT-2 上恢复了短事实准确率同时维持目标层性能，证明检索与参数巩固可以互补。
5. **边界：陈旧记忆失效未解决**：Memora 上的诊断结果为方向性正向但不显著，表明 EVAF 无法处理记忆删除/更新，这是未来工作方向。
