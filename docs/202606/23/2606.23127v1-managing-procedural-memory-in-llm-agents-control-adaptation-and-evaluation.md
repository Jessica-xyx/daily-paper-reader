## 摘要
程序性记忆越来越多地被用于改进LLM智能体在重复性工作场所任务中的表现，但其产生可复用技能的能力仍未被充分理解。我们提出了AFTER基准，包含跨越六个职业角色和22种程序性技能的382个真实企业任务，旨在评估技能如何跨任务、角色和模型主干进行迁移。该基准包括本地改进、跨任务迁移、跨角色迁移和跨模型泛化的受控评估设置。实验表明，程序性记忆在工业工作流中带来持续增益：单次改进轮次使总体性能提升3.7-6.7个百分点，而从多样化多模型执行轨迹演化而来的技能实现了73.1%的跨模型测试准确率，优于所有单模型轨迹来源。我们进一步发现，某些技能能够广泛泛化到不同任务和模型，而另一些则专门化为特定角色的工作流，并在迁移时失去效力。这些结果为在生产智能体平台上构建、评估和部署程序性记忆系统提供了实践指导。

## Abstract
Procedural memory is increasingly used to improve LLM agents on recurring workplace tasks, yet its ability to produce reusable skills remains poorly understood. We introduce AFTER, a benchmark of 382 realistic enterprise tasks spanning six professional roles and 22 procedural skills, designed to evaluate how skills transfer across tasks, roles, and model backbones. The benchmark includes controlled evaluation settings for local improvement, cross-task transfer, cross-role transfer, and cross-model generalization. Experiments show that procedural memory delivers consistent gains in industrial workflows: a single refinement round improves aggregate performance by 3.7-6.7 points, while skills evolved from diverse multi-model execution traces achieve 73.1% cross-model test accuracy, outperforming all single-model trace sources. We further find that some skills generalize broadly across tasks and models, whereas others become specialized to role-specific workflows and lose effectiveness under transfer. These results provide practical guidance for building, evaluating, and deploying procedural memory systems in production agent platforms.

## 6. 主要结论与发现

1. **静态技能增益**：平均M2提升+2.8个百分点，弱模型受益更大（如Gemma 4 E4B在GenAI上+14.2）；LLM生成技能（G）常优于手写技能（H）。
2. **单轮精炼有效**：单次Codex求导使聚合性能额外提升+3.7至+6.7点，大模型提升更稳定。
3. **多样经验优于窄经验**：从5个不同来源轨迹（窄：1个）演化出的技能在测试集上M1提升显著（Hermes：+18.0 vs +0.1）；训练集过拟合常见（如EvoSkill训练集+14.9但测试集-2.7）。
4. **跨模型迁移**：混合多模型轨迹演化的技能测试准确率73.1%，比最佳单源模型（Sonnet 4: 59.4%）高13.7个百分点；弱模型轨迹反而提供更强迁移信号。
5. **跨角色迁移局限**：技能会自然专化于角色特定工作流，同一技能在不同角色间迁移时性能下降（pdf技能从PM→DS损失-4.8点到-7.5点）。
6. **Token效率提升**：演化技能通过前置程序性知识减少推理时探索，总token降低16%~62%，成本相应下降。

## 7. 优点

- **系统性基准设计**：AFTER是首个融合角色结构、技能标注和可控迁移划分（跨任务/跨角色/跨模型）的基准，填补了现有技能基准（SkillsBench）和通用智能体基准（GAIA、SWE-bench）之间的空白。
- **分离技能质量与检索质量**：固定任务-技能映射，使演化获得清晰优化信号，同时允许后续独立研究检索问题。
- **控制变量严谨**：通过E VOLUTION 接口统一管理轨迹、版本和评估，保证框架间公平比较；反射器消融实验排除了特定智能体的偏差。
- **多维度实证**：从静态→单轮→多轮演化→跨模型→跨角色→效率，层层递进，覆盖完整评估链条，结论扎实。
- **开放资源**：基准在HuggingFace公开发布，代码和技能工件可复现。
- **实践指导意义**：揭示了多样经验对泛化的关键作用以及专化风险，指导生产部署中如何平衡广度与深度。

- **基准覆盖范围有限**：仅聚焦技术行业角色和企业办公任务，未覆盖医疗、法律、科研等领域；22项技能排除了开放式创意或对话任务，结论限于程序性、工具使用导向的工作流。
