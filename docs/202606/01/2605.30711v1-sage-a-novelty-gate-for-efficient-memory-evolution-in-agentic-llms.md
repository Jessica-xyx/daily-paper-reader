## 摘要
智能大型语言模型必须持续决定新提取的事实是否应被添加、与已有记忆合并或忽略，然而先前的工作更多关注检索和存储，而非原则性的写入侧控制。我们将记忆演化视为一个新颖性检测问题，并提出SAGE（一种基于球面自适应门控的记忆演化机制），该机制利用基于von Mises-Fisher分布的密度估计器对记忆嵌入上的候选事实进行评分，并通过追踪记忆存储几何结构的自适应阈值对其进行路由。SAGE将清晰的新颖事实解析为添加操作，清晰的冗余事实解析为无操作，仅将不确定案例发送至LLM进行合并步骤，从而减少了昂贵的写入时推理。在LoCoMo上，SAGE在所有七个开源权重骨干对比中均取得了相较于Mem0的最佳平均token-F1分数，而在GPT-4o-mini上，其添加阶段API成本降低了3.4倍，添加阶段延迟降低了2.5倍，且平均评估分数差距很小。作为A-Mem的即插即用二值门控，SAGE在五个模型上跳过了约16-18%的LLM调用，且在开源权重骨干上质量变化极小。这些结果表明，新颖性感知的写入控制是提升长期智能记忆质量与系统效率的实用杠杆。

## Abstract
Agentic LLMs must continuously decide whether newly extracted facts should be added, merged with existing memories, or ignored, yet prior work has focused more on retrieval and storage than on principled write-side control. We frame memory evolution as a novelty-detection problem and propose SAGE, a Spherical Adaptive Gate for memory Evolution that scores candidate facts with a von Mises-Fisher-based density estimator over memory embeddings and routes them with an adaptive threshold that tracks memory-store geometry. SAGE resolves clearly novel facts as ADD, clearly redundant facts as NOOP, and sends only uncertain cases to an LLM merge step, reducing expensive write-time reasoning. On LoCoMo, SAGE achieves the best average token-F1 against Mem0 on all seven open-weight backbone comparisons, while on GPT-4o-mini it reduces add-phase API cost by 3.4$\times$ and add-phase latency by 2.5$\times$ with only a small average judge-score gap. As a drop-in binary gate for A-Mem, SAGE skips roughly 16-18% of LLM calls across five models with minimal quality change on open-weight backbones. These results suggest that novelty-aware write control is a practical lever for improving both memory quality and system efficiency in long-term agentic memory.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：现有智能体LLM长期记忆系统主要关注检索和存储（如嵌入模型、向量索引、知识图谱），而对写入侧控制（写决策）缺乏原则性研究。实际上，写决策更为关键：错误写入（重复、与无关事实合并、过早删除）会劣化下游查询。当前系统（如Mem0、A-Mem）将写决策全部交由LLM在每次写入时作出，导致写路径成为主要成本来源。
- **核心问题**：如何设计一个轻量、高效的写控制机制，既能保持记忆质量又能大幅降低写入阶段的LLM调用次数？
- **整体含义**：将记忆演化形式化为一个**新颖性检测问题**，提出SAGE（Spherical Adaptive Gate for memory Evolution），用闭式（closed-form）方法区分明显新颖、明显冗余和不确定的候选事实，仅将不确定案例交给LLM处理，从而在质量和效率之间取得更好权衡。

## 2. 方法论：核心思想、关键技术细节、公式与算法流程

### 核心思想
- 将内存写决策建模为新颖性检测：对每个候选事实c，计算其相对于当前记忆集合M的**新颖性评分**ν(c)。
- 根据ν(c)与自适应阈值τt的关系，自动路由到**ADD**（新颖）、**NOOP**（冗余）或**UPDATE**（不确定）。
- 仅UPDATE情况需要LLM进行合并推理，ADD和NOOP均无LLM调用。

### 关键技术细节与公式
1. **嵌入空间与归一化**：每个记忆项和候选事实都由句子嵌入模型映射到单位超球面S^(d-1)上，即L2归一化向量。
2. **von Mises-Fisher（vMF）核密度估计**：对候选c，定义核函数K_κ(c, m_i)=exp(κ m_i^T c)。则vMF聚合得分：  
   s_vMF(c|M) = (1/κ) log (1/N Σ_i exp(κ m_i^T c))。  
   该得分可解释为所有记忆与c的余弦相似度的log-mean-exp，反映整体支撑程度。取值范围为[-1,1]。
3. **新颖性评分**：ν(c)= (1 - s_vMF(c|M)) / 2，使越大越新颖。
4. **自适应浓度参数κ**：根据记忆向量的平均合成长度R̂估计，使核宽度适应记忆分布的稠密程度。
5. **自适应阈值**：利用PCA投影后计算记忆集的体积V_t和密度ρ_t = N_t / V_t。阈值公式：τ*_t = τ_min + τ_0 e^{-λ ρ_t}，并用指数移动平均（EMA）平滑得到τ_t。
6. **不确定性带**: margin δ定义宽δ的区间[τ_t, τ_t+δ)。候选落在其中时触发LLM（UPDATE）；否则根据上下界直接ADD或NOOP。

### 算法流程（文字描述）
1. 用户交互 → LLM事实提取 → 得到候选事实c。
2. 将c嵌入并归一化。
3. 若记忆集为空，直接ADD；否则计算ν(c)。
4. 根据当前ρ_t计算τ_t（含EMA平滑）。
5. 判断：ν(c) ≥ τ_t+δ → ADD（无LLM）；ν(c) < τ_t → NOOP（无LLM）；τ_t ≤ ν(c) < τ_t+δ → UPDATE（调用LLM进行合并）。
6. 写入/更新记忆库，内存几何更新，ρ_t更新。

- **基准**：主要对比Mem0和Mem0g（实体关系图增强版本）。另有独立A-Mem对比实验（将SAGE作为其前置门控）。

## 6. 主要结论与发现

1. **质量优势**：SAGE在所有7个开源骨干上取得平均token-F1最高，B1在6/7骨干上最高，Judge在4/7骨干上最高（且超过Mem0 in 6/7）。在GPT-4o-mini上多跳和开放域类别率先或持平，但单跳和时间类别略弱。
2. **效率优势**：SAGE实现零路由LLM调用，仅需对少量（2.7-10.6%）不确定候选调用LLM进行合并。总写端LLM调用减少29-42%（排除一个异常骨干后均值32%）。
3. **A-Mem门控效果**：作为前置NOOP门控，可在四个开源模型上以<0.5%的token-F1变化跳过约16-18%的写端LLM调用，在GPT-4o-mini上牺牲2% Judge但节省更大计算。
4. **自适应阈值优于固定阈值**：固定阈值需要针对不同骨干调整，否则急剧退化，自适应阈值自动适应记忆几何，鲁棒性更好。

## 7. 优点

- **新颖视角**：首次将记忆演化形式化为新颖性检测问题，并设计原则性的闭式门控，替代昂贵的LLM路由。
- **理论扎实**：基于vMF分布密度估计和自适应阈值的推导符合超球面几何特性，且参数自适应于记忆密度。
- **效率与质量兼顾**：不牺牲质量即可大幅降低写端计算；在GPT-4o-mini上以微小质量代价换取数倍成本降低。
- **即插即用**：可作为轻量级前置门控嵌入任何现有记忆系统（如A-Mem），无需修改内部逻辑。
- **实验设计公平**：骨干匹配、共享事实提取流程和检索栈，只改变路由控制器。

## 8. 不足与局限

- **基准覆盖有限**：仅使用LoCoMo（英文多会话对话），未在LongMemEval、工具使用、多语言或多模态场景上评估。
- **功能缺失**：当前SAGE不支持DELETE或记忆压缩机制；vMF仅能路由ADD/UPDATE/NOOP，不能直接删除冗余记忆。
- **嵌入模型局限性**：依赖L2归一化句子嵌入，语义相近但嵌入不同的表述可能被错误保留或遗漏。
- **没有与最新RL方法对比**：未与Memory-R1、Mem-α等基于强化学习的记忆管理器进行实验对比。
- **固定阈值门控在GPT-4o-mini上质量下降略大**（Judge下降2%），对更强模型需进一步研究。
- **超参数敏感**：虽然自适应阈值优于固定阈值，但δ（不确定带宽）仍可调，需权衡调用次数和质量。
