## 摘要
世界模型为长视野LLM智能体提供了一种原则性的方法，使其具备预见能力：即在执行前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至损害下游决策。本文介绍了WorldEvolver，一种自我演化的世界模型框架，它在冻结下游智能体和所有模型参数的同时，修改其部署时的上下文。WorldEvolver集成了三个模块：(i) 情景记忆，通过基于检索的模拟利用真实行动转换；(ii) 语义记忆，从预测-观测不匹配中提取持久的启发式规则；(iii) 选择性预见，在将低置信度预测整合到智能体推理上下文之前对其进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上衡量世界模型预测准确率，在AgentBoard上衡量下游智能体成功率。大量实验表明，WorldEvolver在三个骨干网络上实现了最高预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修正能同时提升预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 2. 方法论

- **核心思想**：将经历抽象为两种记忆——**情景记忆**（Episodic Memory）存储具体过渡经验，**语义记忆**（Semantic Memory）从预测-观测不匹配中提取持久启发式规则；再通过**选择性预见**（Selective Foresight）过滤低置信度预测，避免误导智能体。

- **关键技术细节**：
  - **情景记忆**：\(M_E^t = \{(o_i, a_i, o_{i+1})\}_{i<t}\)，根据候选动作的Jaccard相似度检索top-\(k_{ME}\)个已执行过渡，将检索结果以原始文本形式注入世界模型上下文。
  - **语义记忆**：当预测\(\hat{o}_{t+1}\)与真实观测\(o_{t+1}\)经分解（factorization）为三元组后不一致时，LLM批评器（critic）生成候选规则，每条规则附带证据分数\(e_i\)，仅保留正分规则。支持mini-batch聚合更新（batch size \(k_{MS}\)）。
  - **选择性预见**：计算预测输出的几何平均token概率\(\ell_t = \frac{1}{n}\sum \log p_\theta(y_i)\)，转换为置信度\(q_t = \exp(\ell_t)\)，仅当\(q_t \geq \tau\)时预测才暴露给智能体，否则为空（\(\emptyset\)）。

- **算法流程**（Algorithm 1）：
  1. 从冻结策略\(\pi_\theta\)采样草稿动作\(a_t^{(0)}\)。
  2. 根据\(a_t^{(0)}\)检索情景记忆得到\(k_{ME}\)条过渡。
  3. 用世界模型\(W_\theta\)在上下文（含情景/语义记忆）下预测\((\hat{o}_{t+1}, q_t)\)。
  4. 选择性过滤：若\(q_t<\tau\)则\(F_t=\emptyset\)，否则\(F_t=\hat{o}_{t+1}\)。
  5. 智能体基于\((s_t, F_t)\)采样最终动作\(a_t = a_t^{(1)}\)。
  6. 若最终动作与草稿不同，用\(a_t\)重新查询世界模型得到对齐的\(\hat{o}_{t+1}\)。
  7. 执行动作，获取真实观测\(o_{t+1}\)。
  8. 追加过渡到情景记忆：\(M_E^{t+1} = M_E^t \cup \{(o_t, a_t, o_{t+1})\}\)。
  9. 若\(\hat{o}_{t+1} \neq o_{t+1}\)（因子分解后），生成规则更新语义记忆。

## 7. 优点

- **训练自由**：不更新任何模型参数，仅通过外部记忆上下文演化，避免梯度计算和灾难性遗忘。
- **记忆机制互补**：情景记忆提供具体经验（剥削），语义记忆从失败中提炼抽象规则（探索），符合经典认知分类。
- **选择性过滤**：基于置信度的门控有效防止低质量预测误导智能体，实验证明弱骨干受益更大。
- **在线持续学习**：情景和语义记忆跨轨迹累积，使后续试次能利用前期证据，提升长期性能。
- **实验设计全面**：涵盖预测准确率和下游规划两个维度，跨多种骨干、策略和环境，消融、超参分析完整。
