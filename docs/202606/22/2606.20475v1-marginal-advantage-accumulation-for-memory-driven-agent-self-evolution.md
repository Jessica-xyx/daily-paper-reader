## 摘要
在批量式痕迹蒸馏中，同一记忆操作可能在不同批次间收到矛盾反馈。现有方法缺乏跨批次、操作级别的证据积累机制，无法区分稳定有效的操作与偶然命中。本文将该需求形式化为两个结构条件——可对齐性与可比性，并提出边际优势积累（MAA）。MAA构建差分信号使其跨批次可比，通过指数移动平均逐操作积累带符号证据，并通过语义身份合并确保跨批次可追溯性。作为一种后处理架构，MAA在4个基准和4个目标模型的16个设置中的14个上取得最佳结果，持续优于现有批量级蒸馏基线，并在多数设置中匹配或超越在线替代方案，同时将优化阶段的令牌消耗减少约75%。

## Abstract
In batch-style trace distillation, the same memory operation may receive contradictory feedback across different batches. Existing methods lack a cross-batch, operation-level evidence accumulation mechanism, making it impossible to distinguish stably effective operations from accidental hits. This paper formalizes the requirement as two structural conditions, alignability and comparability, and proposes Marginal Advantage Accumulation (MAA). MAA constructs differential signals to make them comparable across batches, accumulates signed evidence per operation via EMA, and ensures cross-batch traceability through semantic identity merging. As a post-processing architecture, MAA achieves the best results in 14 out of 16 settings across 4 benchmarks and 4 target models, consistently outperforming existing batch-level distillation baselines and matching or surpassing online alternatives in most settings, while reducing optimization-phase token consumption by approximately 75%.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：在大语言模型驱动的自主智能体中，记忆驱动的自我进化是关键问题。批量式痕迹蒸馏（batch-style trace distillation）中，同一记忆操作在不同的执行批次中可能收到矛盾的反馈（例如在一个批次中有益、在另一个批次中有害），但现有方法缺乏跨批次、操作级别的证据积累机制，无法区分“稳定有效的操作”与“偶然命中的操作”。这使得蒸馏决策容易受局部批次干扰，导致记忆质量不稳定。
- **核心问题形式化**：作者将所需机制抽象为两个结构条件——
    - **可对齐性（Alignability）**：系统必须能在不同批次中识别语义等效的操作，并将其合并为同一累积单元，否则无法跨批次跟踪。
    - **可比性（Comparability）**：不同批次的任务分布不同，同一操作在不同批次产生的信号需要具有一致的尺度，以便进行有意义的累加。若信号被批次特性主导，则跨批次聚合无效。
- **整体含义**：MAA 通过设计满足上述两个条件的证据累积机制，实现了在不依赖在线环境重执行的情况下，利用离线痕迹进行更可靠的记忆优化，从而提升智能体自我进化的性能和效率。

## 2. 提出的方法论

### 核心思想
将用户感知的“优化建议”转化为可寻址的原子操作（operation, op），这些操作在不同批次间具有不变的身份；在此基础上构建尺度一致的累积信号，使得同一操作跨批次的证据可以相加聚合。通过差分构造隔离批次难度影响，通过指数移动平均（EMA）放大方向一致的信号、抵消方向交替的信号。

### 关键技术细节

1. **可寻址记忆库与语义身份合并**：
   - 记忆库 `M` 被抽象为一个具有稳定 ID 的文本块集合，每个操作通过 `(type, anchor_id, content_embedding)` 元组标识。
   - 当新候选操作的语义相似度（余弦相似度）超过阈值 `τ=0.85` 时，合并到同一累积单元，避免因措辞差异导致身份碎片化（满足 **可对齐性**）。

2. **边际优势构造**：
   - 使用 LLM 评分通道估计当前基线记忆 `M` 在批次 `B` 上的期望效用 `u(M,B) ∈ [0,100]`。
   - 对于每个候选操作 `i`，计算 **差分信号**：`δ_i = u(M_i, B) - u(M, B) ∈ [-100, 100]`，其中 `M_i` 是应用该操作后的假想状态。
   - **差分的作用**：将绝对评分转化为相对于当前基线的局部比较，缓解不同批次难度分布差异，使信号满足 **可比性**。虽然数值幅度可能噪声大，但符号方向（sign(δ)）在扰动下高度稳定（符号一致性 >88.7%），且与真实增益方向显著对齐（符号准确率 ≥61.5%）。

3. **跨批次累积与 EMA**：
   - 每个操作 `k` 维护一个累积量 `m_k`，通过 EMA 更新：`m_{k,t} = β·m_{k,t-1} + (1-β)·δ_{k,t}`，并进行偏差校正得到 `m̂_{k,t}`。
   - EMA 的优势：保留幅度信息（区分“显著改进”与“轻微改进”），指数衰减适应记忆状态的非平稳性（旧信号权重降低），计算复杂度 O(1)。
   - 默认 `β=0.9`，有效窗口约 10 步。

4. **候选池管理与更新预算**：
   - 候选池 `P` 跨步骤保留，新操作经语义去重后加入。按 `m̂_k` 降序管理：低于下界 `m_floor=-50` 的淘汰；超过池大小上限（20）的截断；超过最大年龄（max_age=10）未选中的淘汰。
   - 每步更新上限 `k_t` 随训练进度线性衰减（`r_max=0.4` → `r_min=0.1`），实现前期探索、后期保守。
   - 最佳检查点回滚：保存验证集最优的记忆快照，避免累积有害操作。

### 算法流程（文字说明）
- 初始化：加载初始记忆 `M_0`，清空候选池 `P` 和 EMA 状态。
- 训练循环：每个 epoch 内逐批处理——
  1. 采样批次 `B_t` 及对应痕迹。
  2. **Propose 通道**：根据当前记忆和痕迹生成候选操作（仅提议方向，不评分）。
  3. 候选池维护：语义合并、修剪（无效锚点、低分、超龄）。
  4. **Score 通道**：对候选池中每个操作，评估应用后状态的效用 `u(M_i, B_t)`，并与基线效用 `u(M, B_t)` 差分得到 `δ`。
  5. EMA 更新：更新每个操作的累积量和年龄。
  6. Top-`k` 选择：选取 `m̂_k > 0` 且排名靠前的操作应用到记忆。
  7. 淘汰超龄且未被选中的操作。
  8. 每个 epoch 结束：在验证集 `D_val` 上评估，若性能提升则保存为最佳快照；早停。

## 6. 主要结论与发现

- MAA 在 **14/16 个设置** 中取得最佳性能，一致超越批量级蒸馏基线（Reactive Update、Trace2Skill），并在 **多数设置中匹配或超越在线方法 SkillOpt**，尤其在使用强模型（Qwen3.7-Max, GPT-5.4）时完全领先。
- **差分构造是累积增益的主要来源**：Abs-score EMA 仅带来有限提升（0.5-2.4 pp），Counting-δ EMA 大幅提升（3.2-6.1 pp），证明有符号方向性证据的价值。
- **连续幅度提供精细化排序增益**：Continuous-δ EMA 较 Counting-δ EMA 额外提升 1.1-1.6 pp。
- **跨批次证据积累与任务复杂度正相关**：复杂任务（SpreadsheetBench, ALFWorld）上 MAA 较 Reactive 提升更大（+7.1 pp, +4.2 pp），简单任务（HotpotQA）提升较小（+2.7 pp）。
- **在线方法与离线方法的优劣分界**：在强模型上 MAA 超越 SkillOpt；在弱模型+困难任务（ScienceAgentBench）上，因离线代理信号质量下降（α 小），SkillOpt 优于 MAA，说明离线代理评分的有效性受模型能力制约。
- **计算效率**：MAA 优化阶段 token 消耗减少约 75%，训练时间缩短约 5-6 倍，同时达到更优或相当的性能。

## 7. 优点

- **创新性问题形式化**：将跨批次证据缺失问题提炼为可对齐性和可比性两个结构条件，为后续研究提供清晰框架。
- **轻量级后处理架构**：MAA 仅需修改上下文注入的内容（记忆），无需模型训练、无需环境重执行，易于集成到现有系统。
- **差分设计精巧**：通过同批次基线差分消除批次难度干扰，使符号方向信号可靠，且从理论上保证即使代理评分完全失效，系统性能也不会差于无累积基线（降级安全）。
- **EMA 累积的适应性**：指数衰减适应记忆状态的非平稳性；保留幅度信息支持排名；计算高效 O(1)。
- **语义身份合并**：基于轻量嵌入模型合并等效操作，避免身份碎片化，且合并阈值有鲁棒性分析。
- **工程细节完备**：包括候选池管理、top-k 稀疏更新、最佳检查点回滚、冲突解决等，构成完整可部署系统。
- **实验严谨**：多基准、多模型、多种子、消融与机制诊断结合，且限定了 Pilot 调优与正式实验的数据隔离。

## 8. 不足与局限

- **LLM Judge 的系统性偏差**：代理评分通道可能受位置偏差、长度偏差等影响，尽管通过随机打乱候选顺序和差分构造缓解，但残余偏差仍存在（尤其在弱模型和长链任务上）。
- **弱模型+困难任务场景的局限性**：在 ScienceAgentBench + Qwen3.6-Flash/DeepSeek-V4-Flash 上，MAA 被 SkillOpt 超越，说明当代理信号方向准确率 α 较低时，在线实时反馈更有效。MAA 更适合代理模型较强、痕迹质量较高的场景。
- **实验覆盖范围有限**：仅评估 4 个基准，未涉及更广泛领域（如机器人控制、GUI 交互、多智能体协作等）；大部分消融和诊断仅在 Qwen3.7-Max 上进行，未在弱模型上复现。
- **超参数调优依赖**：核心超参数（β=0.9, τ=0.85, m_floor=-50）通过独立 pilot 调优确定，虽然实验表明稳健，但理论指导较少，最优值可能随任务变化。
- **痕迹覆盖衰减风险**：当记忆在单步中发生显著变化时，旧痕迹对更新后状态的覆盖可能下降。文中提及可通过监测 δ 方差异常触发痕迹刷新，但未作为标准流程。
- **未与强化学习等参数优化方法直接对比**：MAA 属于非参数方法，RL 方法（PPO, DPO 等）与 MAA 正交，但文中未进行联合部署实验。
- **可重复性细节**：虽然提供了算法伪代码和部分提示模板，但具体评分通道的 LLM 调用细节、分组批评分策略等依赖 API 服务，可能难以完全复现。
