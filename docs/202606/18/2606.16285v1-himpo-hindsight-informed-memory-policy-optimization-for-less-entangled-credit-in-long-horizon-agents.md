## 摘要
长周期智能体依赖记忆机制压缩交互历史，但优化记忆写入面临独特的信用分配挑战：记忆更新可能因下游工具失败、噪声观测或推理错误而非其自身贡献而受到奖励或惩罚。这种因果纠缠的信用可能导致智能体丢弃有用证据或保留无关信息。我们提出HiMPO，一个后见之明信息记忆策略优化框架，用于在长周期智能体中为记忆写入动作分配更少纠缠的信用。HiMPO首先通过比较同一写入前状态下从先前和更新后的记忆中可恢复的任务相关信息，来估计记忆更新的局部效用。然后，它使用后见相关性作为有界的回顾过滤器，当局部效用未得到目标结果支持时，衰减记忆信用。得到的记忆特定优势仅应用于记忆令牌，而轨迹级奖励则优化智能体其余行为。在基于裁判的开放域任务和客观压缩记忆问答中，HiMPO在保持压缩上下文效率的同时，优于基于记忆和基于强化学习的强基线。受控干预进一步表明，HiMPO减少了由工具引发错误造成的责备泄漏，并提高了记忆更新的归因保真度。

## Abstract
Long-horizon agents rely on memory mechanisms to compress interaction history, but optimizing memory writing faces a distinct credit assignment challenge: a memory update may be rewarded or penalized due to downstream tool failures, noisy observations, or reasoning errors rather than its own contribution. This causally entangled credit can lead agents to discard useful evidence or preserve irrelevant information. We propose HiMPO, a Hindsight-Informed Memory Policy Optimization framework for assigning less-entangled credit to memory-writing actions in long-horizon agents. HiMPO first estimates the local utility of a memory update by comparing the task-relevant information recoverable from the previous and updated memories under the same pre-write state. It then uses hindsight relevance as a bounded retrospective filter that attenuates memory credit when local utility is not supported by the target outcome. The resulting memory-specific advantage is applied only to memory tokens, while trajectory-level rewards optimize the rest of the agent behavior. Across judge-based open-domain tasks and objective compressive-memory QA, HiMPO improves over strong memory-based and RL-based baselines while preserving compressed-context efficiency. Controlled interventions further show that HiMPO reduces blame leakage from tool-induced errors and improves attribution fidelity of memory updates.

## 2. 论文提出的方法论

### 2.1 核心思想

- HiMPO（Hindsight-Informed Memory Policy Optimization）将每个记忆更新视为一个策略控制的写入动作，通过两个互补信号构建记忆特定优势（memory-specific advantage）：**局部反事实效用**（local counterfactual utility）和**后见相关性**（hindsight relevance），然后使用后见相关性作为有界回顾过滤器来衰减可能纠缠的信用。该优势仅应用于 `<mem>` 令牌，其他令牌仍使用轨迹级优势，以保持整体稳定性。

### 2.2 关键技术细节

- **局部反事实效用**：对于写入前状态 \(H_{i,t}\)，将新的记忆 \(m_{i,t}\) 与旧记忆 \(m_{i,t-1}\) 分别替换进相同压缩状态，计算目标答案的对数似然得分，差值 \(\Delta_{i,t}\) 即为该记忆更新的边际效用。首次写入时与空记忆基线对比。最后对每个步长进行标准化得到 \(\hat{\Delta}_{i,t}\)。
- **后见相关性**：给定写入前状态和 oracle 目标结果 \(z^*_i\)，计算所生成记忆令牌的后见似然 \(h_{i,t}\)（平均对数似然）。然后对同一提示组内相同步长的后见似然进行自归一化得到 \(\log\rho_{i,t}\)，表示该记忆更新相对于兄弟更新在目标结果下的后见相关性。
- **门控机制**：通过符号一致门函数 \(G(\hat{\Delta}_{i,t},\log\rho_{i,t})\) 结合两个信号，仅当两者符号一致时允许信用通过，否则抑制。最终记忆优势 \(A^M_{i,t} = G(\cdot) \cdot \hat{\Delta}_{i,t}\)。
- **稳定机制**：① **成功轨迹保护掩码**：若轨迹成功且记忆优势为负，则将其置零，避免战胜轨迹中否定微观信号；② **反向平滑**：使用指数移动平均将下游效用向后传播到早期写前记忆。
- **令牌级策略优化**：将记忆优势仅散射到 `<mem>` 令牌，其他令牌使用轨迹级优势 \(A^T_i\)，使用标准裁剪策略目标（公式16）联合优化。

## 7. 优点

- **创新的信用分配框架**：首次将记忆更新视为状态写入动作，通过局部反事实估计边际效用，并用后见相关性作为有界过滤器，从机制上减少因果纠缠。
- **仅作用于记忆令牌**：不影响其他模块的轨迹级优化，保持了整体训练稳定性和压缩效率。
- **受控干预验证**：通过构造图1中的两种失败场景进行解纠缠分析，提供了超出最终性能的因果性证据，增强了方法的可信度。
- **强兼容性**：HiMPO 可作为插件模块集成到现有的基于组的策略优化框架中，无需额外自回归解码。
- **计算开销低**：额外前向计算仅约 0.5%，在实际系统中几乎可忽略。

## 8. 不足与局限

- **依赖目标结果**：后见相关性需要 oracle 目标结果（或判官提供的正确输出）作为条件。在训练和离线分析中自然可用，但在目标不明确或弱指定的场景中难以应用。这将限制其扩展到模糊奖励环境。
- **非完全因果识别**：自归一化后见比值是实际可行的代理，但并非完整的因果识别程序。未观察到的混淆或状态表示不完善仍可能影响记忆信用。
- **固定记忆写入调度**：HiMPO 遵循每个交互步后强制写入记忆的固定协议，未解决何时触发写入的调度问题。自适应调度可能进一步减少冗余写入、提高效率。
- **评估任务范围有限**：实验主要集中在搜索和问答类长周期智能体。扩展到 GUI 控制、代码代理、具身环境等其他智能体设置仍是未来工作。
- **实验覆盖局限**：仅测试了 Qwen 家族的 4B 和 7B 模型，未在更大模型或不同模型族上验证；仅使用 BCP、FRAMES、Local Wiki Search 三个基准，通用性尚未充分证明。
- **与其他基线比较的公平性**：在 Local Wiki Search 中部分基线行直接引用原论文数字，虽验证了可复现性，但存在环境差异风险。
