## 摘要
智能体记忆研究已快速发展，但几乎完全集中在文本侧：现有基准很少在交互环境中测试智能体何时真正需要记住其看到的内容而非可以写下的内容。我们提出 DMV-Bench（代码：https://github.com/yyyujintang/DMV-Bench），这是首个针对多模态智能体视觉记忆的交互式基准。DMV-Bench 构建于包含 1000 种产品变体的受控家居装饰电商目录之上，其中文本泄漏契约确保每个任务的区分信号仅存在于像素中。在一系列自主购物会话中，每个访问过的产品图像都携带一个独特的预渲染偶然线索，之后要求智能体回忆特定的线索化产品并导航至其 URL。受双重编码理论启发，我们提出 DualMem，一种并行维护视觉和言语编码的记忆架构。在 DMV-Bench 上，DualMem 在 Gemini 2.5 Flash 和 Qwen2.5-VL-7B 上，对于链长 J ∈ {5, 10, 15, 50} 均优于字幕基线和三种最新的多模态智能体记忆系统，其领先优势经受了记忆库大小和编码位置偏差的对照控制，并采用非对称双重编码机制：视觉端到端携带线索，而言语通道发挥较小的查询定位作用。

## Abstract
Research on agent memory has matured rapidly, but almost entirely on the text side: few existing benchmarks ask, in an interactive environment, when an agent genuinely needs to remember what it saw rather than what it could write down. We introduce DMV-Bench (Code: https://github.com/yyyujintang/DMV-Bench), the first interactive benchmark for multimodal-agent visual memory. DMV-Bench is built on a controlled home-furnishing e-commerce catalogue of 1,000 product variants in which a text-leakage contract keeps the discriminative signal of each task in the pixels alone. Across a chain of autonomous shopping sessions, every visited product image carries a unique, pre-rendered incidental cue, and the agent is later asked to recall a particular cued product and navigate to its URL. Inspired by dual-coding theory, we propose DualMem, a memory architecture that maintains a visual and a verbal code in parallel. On DMV-Bench, DualMem outperforms a caption baseline and three recent multimodal agent-memory systems at every chain length J in {5, 10, 15, 50} on both Gemini 2.5 Flash and Qwen2.5-VL-7B, with the lead surviving controls for memory-bank size and encoding-position bias, and an asymmetric dual-coding regime in which vision carries the cue end-to-end while the verbal channel plays a smaller query-grounding role.

- **研究动机**：现有智能体记忆研究几乎完全聚焦于文本记忆，缺乏对多模态（视觉）记忆的交互式评测。许多 web 智能体在长程任务中“看见”了大量图像，但除非一个视觉细节被显式标记为相关，否则不会被编码。当用户后来用视觉细节（例如“那盏底座是三角黄铜的灯”）回溯某个产品时，纯文本记忆无法区分具体产品。因此，需要回答的问题：**哪些任务真正需要智能体记住它看到的，而不是写下的？**
- **背景**：人类记忆常由偶然的视觉线索触发（如包装的颜色），但现有基准（如 VisualWebArena、MemoryArena）要么混淆文本和视觉信号，要么只关注文本，要么不在交互环境中进行跨会话的长程评测。
- **整体含义**：提出首个**交互式、多会话、视觉智能体记忆**基准 DMV-Bench，并设计 DualMem 记忆架构，以隔离和精确测量视觉记忆在长程任务中的表现，推动智能体具备“感知连续性”。

## 2. 论文提出的方法论

### 核心思想
- **DMV-Bench 基准**：在受控的家居电商环境中（1000 种产品变体），每个产品图像预先渲染一个唯一的视觉“偶然线索”（如特定颜色的小物体），**文本泄漏合同（L2-leakage contract）** 确保该线索不出现在标题、描述、alt-text 等任何文本通道中。智能体在自主购物会话中无意间看到这些线索，后续任务要求它根据线索描述（如“那只带有青色睡眠眼罩的产品”）导航回该产品。
- **DualMem 记忆架构**：受双重编码理论（Paivio, 1971）启发，并行维护视觉和言语两路信号，在编码、检索和注入三个阶段均保持双重编码。

### 关键技术细节
- **编码**：每次观察（产品页面）产生四个存储项：原始图像 \(I_i\)、VLM 生成的字幕 \(c_i\)、视觉嵌入 \(v_i\)（用 SigLIP-2 编码图像）、言语嵌入 \(t_i\)（用 SBERT 编码字幕）。
- **检索**：给定查询 \(q\)，同样编码为视觉查询 \(q_v\) 和言语查询 \(q_t\)。对记忆库中每一项 \(i\)，分别计算内积相似度 \(s_v^{(i)} = \langle q_v, v_i \rangle\) 和 \(s_t^{(i)} = \langle q_t, t_i \rangle\)。
- **融合**：对两种相似度进行 min-max 归一化，使它们尺度一致：
  \[
  \tilde{s}_v^{(i)} = \frac{s_v^{(i)} - \min_j s_v^{(j)}}{\max_j s_v^{(j)} - \min_j s_v^{(j)}}, \quad \tilde{s}_t^{(i)} \text{同理}
  \]
  融合得分：\(s^{(i)} = \alpha \tilde{s}_v^{(i)} + (1-\alpha) \tilde{s}_t^{(i)}\)，实验中取 \(\alpha = 0.75\)。
- **注入**：将得分最高的项对应的原始图像 \(I_{e^*}\) 和字幕 \(c_{e^*}\) 一起注入 VLMs 的上下文。

### 高效评估（共享前缀滚动树）
- 为避免重复执行前序会话，构建共享前缀的滚动树：每个会话只运行一次，所有后代复用其记忆快照。深度 \(J\)、分支因子 \(B\) 的树仅需 \(\frac{B^{J}-1}{B-1}\) 次运行，即可产生约 \(B^{J-1}\) 条不同召回路径，节省约 \(J\) 倍开销。

## 3. 实验设计

- **数据集 / 场景**：自建的 DMV-Bench 电商环境，10 类产品（沙发、灯、地毯等）×10 种室内设计风格 ×10 个变体 = 1000 个产品。每个产品图像包含唯一线索，文本无泄漏。
- **基准测试**：DMV-Bench 本身即是评测基准；对比方法包括：
  - **参考基线**：NoMemory（无记忆）、TextOnly（仅产品类别文本）、Caption（VLM 生成的字幕）。
  - **近期多模态外部记忆系统**：WorldMM、MMA、M2A。
  - **DualMem（本文方法）**。
- **VLM 后端**：Gemini 2.5 Flash（通过 API）和 Qwen2.5-VL-7B-Instruct（本地部署）。

- **评价指标**：任务成功率（TSR），即智能体最终导航到正确产品 URL 的比例（确定性匹配）。

## 6. 论文的主要结论与发现

1. **DualMem 在所有链长和两个后端上均领先**：在 \(J=5,10,15,50\) 上，DualMem 的 TSR 均高于 Caption、WorldMM、MMA、M2A。例如在 Gemini 2.5 Flash 上 \(J=5\) 时 DualMem 82.7% vs 第二名 M2A 65.7%。
2. **视觉记忆的必要性**：纯文本基线（NoMemory、TextOnly）在所有设置下 TSR 为 0%，证实 L2-leakage 合同有效，视觉记忆必不可少。
3. **非对称双重编码**：视觉通道端到端承载线索（检索和注入时视觉占主导），言语通道主要起查询定位的辅助作用。α=0.75 最佳，纯视觉 (α=1) 80.1%，纯言语 (α=0) 59.5%。
4. **控制实验揭示稳定性**：DualMem 的 TSR 在记忆库大小和编码位置轴上几乎平坦，而基线则随库增大而衰减、随位置出现漂移，说明优势来自记忆架构本身。
