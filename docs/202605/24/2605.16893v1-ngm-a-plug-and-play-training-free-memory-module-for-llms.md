## 摘要
近期研究引入了条件记忆模块，将知识存储与神经计算解耦，从而实现了更直接的知识访问。与依赖动态计算路径的MoE相比，显式查找提供了一种更高效的知识检索机制。然而，这些方法仍然依赖于学习得到的记忆嵌入，需要额外训练且限制了灵活性。为此，我们提出了N元记忆（NGM），一种无需训练、即插即用的模块，由因果N元编码器和余弦门控记忆注入器组成。因果N元编码器直接平均主干模型的预训练词嵌入来构建N元表示，从而无需从头训练单独的N元嵌入。该设计既不需要额外的记忆表，也不需要检索流水线。余弦门控记忆注入器则使用非参数化的余弦门控和ReLU，将检索到的嵌入调制到上下文表示中。我们在Qwen3系列（0.6B至14B）的八个基准上评估了NGM。NGM将平均性能提升了0.5到1.2个百分点，在代码生成和知识密集型任务上尤其显著（例如，Qwen3-14B在LiveCodeBench上提升+3.0，在GPQA上提升+3.03）。此外，NGM在多模态基准上也提升了性能（例如，Qwen3-VL-2B在MMStar上提升+1.53）。

## Abstract
Recent studies introduce conditional memory modules that decouple knowledge storage from neural computation, enabling more direct knowledge access. Compared to MoE, which relies on dynamic computation paths, explicit lookup provides a more efficient knowledge retrieval mechanism. However, these approaches still depend on learned memory embeddings, requiring additional training and limiting flexibility. To address this, we propose N-gram Memory (NGM), a training-free, plug-and-play module composed of a Causal N-Gram Encoder and a Cosine-Gated Memory Injector. The Causal N-Gram Encoder directly averages the pretrained token embeddings of the backbone model to construct N-gram representations, thereby eliminating the need to train separate N-gram embeddings from scratch. This design requires neither an additional memory table nor a retrieval pipeline. The Cosine-Gated Memory Injector then uses a non-parametric cosine gate with ReLU to modulate the retrieved embeddings into the contextual representations. We evaluate NGM on the Qwen3 series from 0.6B to 14B across eight benchmarks. NGM improves average performance by 0.5 to 1.2 points, with particularly clear gains on code generation and knowledge-intensive tasks (e.g., +3.0 on LiveCodeBench and +3.03 on GPQA for Qwen3-14B). Moreover, NGM also improves performance in multimodal benchmarks (e.g., MMStar +1.53 on Qwen3-VL-2B).

**研究动机**：在Transformer大语言模型（LLM）中，语言建模需要同时处理动态组合式计算和局部静态模式复用（如命名实体、重复标识符、术语、公式短语等）。标准Transformer缺乏原生的知识查找机制，导致模型在推理时通过昂贵的注意力和前馈计算重新构建这些模式。现有研究（如Engram、SCONE、L³、MeKi等）引入显式条件记忆模块，通过训练专门的N-gram嵌入表或扩展嵌入容量来解耦知识存储，但都依赖额外训练参数、专用存储或检索基础设施，灵活性受限。

**核心问题**：**能否在不重新训练、不添加学习记忆表的情况下，让已经训练好的LLM获得局部记忆带来的好处？**

**整体含义**：提出一种即插即用、无需训练的N-gram记忆模块（NGM），直接利用主干模型已有的预训练词嵌入构建局部N-gram表示，并通过非参数余弦门控注入到冻结的解码器表示中，从而在零额外参数、零训练成本下提升模型在局部模式依赖任务上的性能。

## 2. 论文提出的方法论

### 核心思想
将预训练词嵌入空间不仅作为输入接口，也作为可复用的局部记忆源。通过聚合因果窗口内的词嵌入构造多尺度N-gram表示，再利用余弦相似度作为门控信号，将与当前隐藏状态对齐的局部记忆注入到解码器表示中。

### 关键技术细节

**（1）因果N-gram编码器（Causal N-gram Encoder）**  
- 输入：token序列 \(X = \{x_1,...,x_T\}\) 和主干模型词嵌入矩阵 \(E \in \mathbb{R}^{V\times d}\)。  
- 对每个N-gram大小 \(n\)（如2、3），左填充 \(n-1\) 个零向量，然后对每个位置 \(t\) 进行因果平均池化：  
  \[
  g_{t,n} = \frac{1}{n}\sum_{k=0}^{n-1} \tilde{e}_{t-k}
  \]  
  其中 \(\tilde{e}_t\) 是填充后的嵌入。  
- 多尺度表示堆叠成矩阵 \(G_t = [g_{t,n}]_{n\in\mathcal{N}} \in \mathbb{R}^{|\mathcal{N}|\times d}\)。

**（2）余弦门控记忆注入器（Cosine-Gated Memory Injector）**  
- 对给定解码器层 \(l\) 的隐藏状态 \(h_t^l\) 与每个N-gram向量 \(g_{t,n}\) 计算余弦相似度：  
  \[
  s_{t,n} = \cos(h_t^l, g_{t,n})
  \]  
- 可选应用ReLU抑制负对齐更新：\(\tilde{s}_{t,n} = \max(0, s_{t,n})\)。  
- 跨N-gram尺度聚合：\(\tilde{s}_t = [\tilde{s}_{t,n}]_{n}\)，然后通过残差连接注入：  
  \[
  h_t^{l'} = h_t^l + \lambda \tilde{s}_t^\top G_t
  \]  
  其中 \(\lambda\) 是输出缩放常数。

**算法流程**（文字描述）：
1. 基于输入token ID和主干词嵌入矩阵，通过因果平均池化构造多尺度N-gram表示矩阵 \(G\)。  
2. 对齐到当前解码步最后 \(L\) 个位置（兼容KV缓存）。  
3. 对每个位置，计算编码器隐藏状态与N-gram表示的余弦相似度，经ReLU过滤。  
4. 加权求和（跨尺度）得到记忆更新向量，乘以缩放因子后加到原始隐藏状态上。

**设计特点**：无需额外记忆表、无需检索索引；复杂度线性于长度和维度；可即插即用于已冻结模型。

## 8. 不足与局限

1. **表示局限性**：因果N-gram编码器使用词嵌入的算术平均，是顺序不敏感的袋式近似，可能对顺序敏感或非组合短语产生误导。  
2. **门控启发式**：余弦门控和固定输出尺度 \(\lambda\) 是固定启发式，非上下文自适应，导致部分任务（如IFEval）性能下降。

4. **实验覆盖局限**：仅验证于Qwen3系列，未在Llama、DeepSeek等其他主流系列上测试，泛化性有待进一步验证。
