## 摘要
部分可观测性下的强化学习需要决定保留哪些信息，但大多数基于记忆的方法并未明确建模符号观测的短期到长期迁移。我们在时序知识图谱记忆设置中研究这一迁移过程，并将其形式化为神经符号的基于价值决策问题：对于每个观测到的三元组，智能体在长期插入前选择保留或丢弃。为处理可变大小的短期缓冲区，我们采用基于项目的Q学习设计，包含共享参数以及在连续步骤中匹配项目上的实用时序差分更新。在长期记忆容量为128的RoomKG基准上，学习到的迁移决策优于符号和神经基线，包括带有时间标注的符号基线和基于历史的LSTM/Transformer基线。在迁移策略消融实验中，轻量级的局部短期变体表现最佳，而步骤级行为表明该策略保留与导航和查询相关的事实，同时丢弃价值较低的候选事实，从而在记忆约束下支持显式且可解释的记忆决策。

## Abstract
Reinforcement learning under partial observability requires deciding what information to retain, yet most memory-based approaches do not explicitly model short-term-to-long-term transfer of symbolic observations. We study this transfer process in a temporal knowledge-graph memory setting and cast it as a neuro-symbolic value-based decision problem: for each observed triple, the agent chooses whether to keep or drop it before long-term insertion. To handle variable-sized short-term buffers, we use a per-item Q-learning design with shared parameters and a practical temporal-difference update over matched items across consecutive steps. On the RoomKG benchmark at long-term memory capacity 128, learned transfer decisions outperform symbolic and neural baselines, including symbolic baselines with temporal annotations and history-based LSTM/Transformer baselines. Across transfer-policy ablations, a lightweight local short-term-only variant performs best, and step-level behavior shows that the policy keeps navigation- and query-relevant facts while discarding lower-value candidate facts, supporting explicit and interpretable memory decisions under memory constraints.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：在部分可观测的强化学习（POMDP）中，智能体需要决定保留哪些信息，但现有基于记忆的方法大多没有显式建模符号观测（如知识图谱三元组）从短期记忆向长期记忆的迁移过程。
- **背景**：人类在部分可观测环境中会选择性将即时经验转移到长期记忆，而现有方法（如DRQN、外部记忆网络）通常使用隐式潜在状态更新，不仅难以解释，也缺乏对记忆内容显式的“保留/丢弃”决策。
- **研究目标**：在时序知识图谱记忆设置中，将短期→长期记忆迁移建模为一个神经符号的基于价值的决策问题。每个观测到的三元组在插入长期记忆前都需做出独立的保留/丢弃决定，从而在记忆容量约束下实现可解释且有效的记忆管理。

## 2. 方法论

### 核心思想
- 将记忆迁移视为一个强化学习问题：智能体在每一时间步接收一组可变数量的短期记忆项（RDF三元组），对每个项输出一个二元动作（keep/drop），之后这些动作影响长期记忆的内容，最终通过对下游问答任务（QA）的奖励来学习迁移策略。
- 采用**per-item Q-learning**框架，以处理可变大小的短期记忆集合：使用共享参数的Q函数为每个短期记忆项独立输出两个Q值（kep/drop）。这样即使短期集合大小随时间变化，模型也能在不固定动作维度下工作。

### 关键技术细节
- **状态表示**：将当前记忆状态（包含短期和长期记忆）转换为图结构（节点和边，带时间标注特征），通过图神经网络（GNN）编码器获得节点嵌入。每个短期记忆项由其头尾节点的拼接表示。
- **Q值输出**：对每个短期项，通过一个小的MLP头输出2维Q值（分别对应keep和drop）。
- **动作选择**：每个短期项独立使用ϵ-贪婪策略选择保留或丢弃。
- **训练**：采用类似DQN的离线训练，存储转移样本（状态、动作向量、奖励、下一状态）。由于短期记忆集合大小可变且无序，采用**随机匹配策略**：在相邻时间步之间，对短期项进行随机配对并计算TD误差，损失为所有匹配项的平均TD平方误差。使用目标网络、经验回放等标准技巧。
- **非迁移组件固定**：为隔离迁移策略的影响，问答、探索和驱逐策略均使用固定的符号启发式（如MRU、LRU等），使性能差异可直接归因于迁移决策。

### 算法流程（文字说明）
1. 初始化在线网络参数θ、目标网络参数\bar{θ}、经验回放池D。
2. 每一环境步t：
   - 构建当前记忆状态 M_t = (M_short_t, M_long_t)。
   - 对每个短期项i，用ϵ-贪婪从Q_θ(M_t, i, ·)选择动作a_{t,i}。
   - 执行迁移决策，获得奖励r_t、下一状态M_{t+1}和终止标志d_t。
   - 将转移 (M_t, a_t, r_t, M_{t+1}, d_t) 存入D。
   - 若达到热启动步数，从D采样小批量转移。
   - 对每个采样转移b，计算ℓ_b = min(|M_short_b|, |M_short_{b+1}|)，随机匹配前后步的短期项，对每个匹配j计算 y_{b,j} = r_b + γ(1 - d_b) max_a Q_{\bar{θ}}(M_{b+1}, j, a)，损失 L = E[ (1/ℓ_b) Σ_j (Q_θ(M_b, j, a_{b,j}) - y_{b,j})² ]。
   - 定期更新目标网络。

## 3. 实验设计

### 数据集/场景
- **RoomKG基准**：一个部分可观测的符号知识图谱环境，7×7网格（49个房间），18个静态物体，18个运动物体，36个内墙（周期性变化），每回合100步。
- 长期记忆容量固定为128（主要实验设置）。
- 使用标准train/test划分（同一动力学但不同查询顺序）。

### 对比方法
- **符号基线**（带时间标注）：
  - Always-Transfer：所有短期项都转移。
  - Novel-Only：只转移长期记忆中不存在的项。
  - Random-Transfer (p=0.5)：每个短期项以50%概率转移。
- **端到端神经基线**（RoomsKG先前工作）：
  - LSTM + Always-Transfer
  - Transformer + Always-Transfer
- **学习迁移策略变体**（DQN + 时间RDF）：
  - 编码器：GCN、R-GCN、StarE-GNN
  - 策略模式：Local-STM（仅短期输入，局部每个项独立决策）、Local-Full（全上下文输入，局部决策）、Global-STM（短期输入，全局共享决策）、Global-Full（全上下文输入，全局共享决策）

### 主要指标
- 每回合QA正确答对数量（越高越好）。
- 报告5个随机种子上的均值和标准差。
